# honeyPot


#Game Engine honeyPot
#Company: Hungry Bear Gaming

#Setup

--Build--
1) run npm install
2) to build run gulp build
3) HoneyPot folder is created in the dist folder
4) the honeypot folder contains vendor.js and app.js and an image folder
5) copy the honeyPot folder to the dist folder in the game
5) add index.htm the 2 script files are
	<script src="honeypot/vendor.js"></script>
	<script src="honeypot/app.js"></script>
