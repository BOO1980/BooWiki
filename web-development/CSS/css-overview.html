CSS Overview

  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • CSS Stands for Cascading Style Sheets
  • CSS saves a lot of work. It can control the layour of multiple web pages all at once
  • CSS exteranl style shees are stored in CSS files
  • HTML was never intended to contain tags for formatting a web page, to add colour and fonts the world wide web consortium (W3C) created CSS
  • A CSS rule consists of a selector and a declaration block.
